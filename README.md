<h1>Meu exercícios de SQL</h1>
Aqui é onde eu publico meus exercícios de práticas de SQL criando esquemas de bancos de dados e simulando consultas reais.
Sempre busco resolver questões que acho interessantes no codeforces, beecrowd, livros, ou qualquer outra plataforma que tenha questões interessantes.
<br>
<h2>Como funciona?</h2>
É bem simples! Eu simplesmente pesquiso questões de SQL que eu considero interessante e simplesmente as resolvo a minha maneira, às vezes eu busco ajuda,
mas na maioria das vezes a ideia é se desafiar e resolver de uma forma criativa e simples. Busco resolver questões de livros e de plataformas como codeforces
beecrowd e entre outros.
